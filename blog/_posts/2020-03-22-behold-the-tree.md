# Behold The Tree

```
Behold the tree.
It stands calmly between the sun
and the deep darkness of
the world which lies beneath.

It is wise.
Remembering, observing,
growing, teaching.

It teaches its children
through the fungal underworld.

It will teach you as well
if you can speak the language.
```
